Professional Summary

I'm a Python-first backend engineer who thinks about software architecturally before syntactically — bounded domains, clean layers, and data flow that's easy to reason about. My primary stack is FastAPI + SQLAlchemy + Alembic, deployed behind Nginx on Linode VPS infrastructure.

I also mentor junior engineers on a parallel TypeScript stack I built as a one-to-one mirror of my Python practices — TSOA for FastAPI's auto-Swagger, Prisma for Alembic-style migrations, matching lint and pre-commit tooling. Beyond the code, I walk them through Git workflows and the production lifecycle: Linode, Nginx, SSH keys, and least-privilege Linux users — so by their first deploy they understand both how the request reaches the app and who's allowed on the box.
