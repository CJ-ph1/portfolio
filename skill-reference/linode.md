# Linode

## 🌐 Web Server Layer
- **Nginx** – web server and reverse proxy used to serve applications and route incoming traffic

---

## ⚙️ Process Management Layer
- **PM2** – application process manager (commonly for Node.js apps), keeps apps running and restarts them if they crash  
- **systemd** – Linux service manager used to control system services and manage startup processes

---

## 📁 File System Navigation
- **File navigation** – basic Linux commands for moving around and inspecting the system (cd, ls -la, etc.)

---

## 🔐 User & Access Control
- **Authorization** – creation and management of users and groups for system access control  
- **File permissions** – defining who can read, write, or execute files and directories

---

## 🔑 Security & Authentication
- **SSH keys** – secure method for accessing servers and authenticating with remote repositories without passwords
